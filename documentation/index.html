Input Area

Defines a rectangular area in the window. Input areas are used for creating buttons, checkboxes and textboxes.

Input Circle Area

Defines a circular area in the window. Input circle areas are used for creating radio buttons.

Input Font

Input fonts define a set of font properties for buttons, checkboxes, textboxes and radio buttons.

Input Border

Input borders define the color, width and selected color of inputs. The selected color field is used as the the checked color for checkboxes and radio buttons. For buttons, the selected color has no effect.

Textbox Text

Defines all of the properties for the text to be displayed in a textbox.

Textbox

Textboxes are rectangles that that store a string.

Checkbox

Checkboxes are small rectangular boxes that store a boolean value.

Radio

Radio buttons are circular buttons that store a boolean value, only one in a radio group can be true however.

Radio Group

Radio groups are a list of radio buttons. Each group can only have one radio button that is checked.

Button

Primarily used for forms, a button allows for form actions to be run through a screen click.

Form

Contains a lists of buttons, checkboxes, textboxes and radio button groups. Very useful for updating, drawing and clearing multiple buttons, checkboxes, textboxes and radio button groups at once.

Form Action

Defines the supported form actions by SplashKit UI.

Parameters
Name Description
CLEAR Clears the form.
NONE Just looks pretty.
SUBMIT Submits the form.

Text Alignment

Defines the alignment of text in textboxes.

Parameters
Name Description
LEFT_ALIGNED Aligns the text to the left.
CENTER_ALIGNED Aligns the text to the center.
RIGHT_ALIGNED Aligns the text to the right.
