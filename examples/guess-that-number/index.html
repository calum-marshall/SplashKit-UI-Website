Guess That Number Program

This program was based off the 4.2C Task.
A random number between 0 and 100 is radomly selected.
A user then guesses the number using the textbox to enter the guess.
A user submits the guess by clicking on the guess button or by pressing the return key.
A message will appear telling the user if the guess is higher, lower or correct.
The game will restart after 3 seconds once the user gets the correct number.
