Guess That Number

A remake of 4.2C, Guess That Number.
A number is randomly chosen between 0 and 100. The user then tries to guess this number using a textbox as input. Prompts are given based upon wether the textbox value is greater than, less than or equal to the number. This program demonstrates the use of a form, a number only textbox, a button, clearing a form and returning a value from a form.
