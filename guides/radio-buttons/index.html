Radio Buttons

Before you begin, make sure your directory has been setup as described in the installation guide.

In this guide we will create a radio button group. The program will ask a user to select their favourite sport from a list of radio buttons.
To get started let's create the input_font and the input_circle_area for each radio button and also the list of radio buttons for the radio button group.

input_font program_font = new_input_font("Calibri", 30, COLOR_BLACK);

input_circle_area radio_1_area = new_input_circle_area(40, 80, 8);
input_circle_area radio_2_area = new_input_circle_area(40, 110, 8);
input_circle_area radio_3_area = new_input_circle_area(40, 140, 8);
input_circle_area radio_4_area = new_input_circle_area(40, 170, 8);

vector<radio> radio_buttons_to_add;
After creating the above artefacts, we can create the actual radio buttons and insert them into a radio button group as follows.
radio radio_1 = new_radio("radio_1", "AFL", radio_1_area, program_font);
radio_buttons_to_add.push_back(radio_1);
radio radio_2 = new_radio("radio_2", "Cricket", radio_2_area, program_font);
radio_buttons_to_add.push_back(radio_2);
radio radio_3 = new_radio("radio_3", "Soccer", radio_3_area, program_font);
radio_buttons_to_add.push_back(radio_3);
radio radio_4 = new_radio("radio_4", "Tennis", radio_4_area, program_font);
radio_buttons_to_add.push_back(radio_4);

radio_group radio_group_1 = new_radio_group("radio_group_1", radio_buttons_to_add);
Now that we have our radio button group with our buttons created we have to draw and update the group as follows.
while (!quit_requested())
{
  process_events();
  clear_screen(COLOR_WHITE);
  draw_text("Which is your most favourite sport?", COLOR_BLACK, "Calibri", 45, 20, 20);
  draw_radio_group(radio_group_1);
  update_radio_group(radio_group_1);
  refresh_screen();
}
You now have four fully functional radio buttons (as long as you followed the guide correctly).
You can access the selected value of a radio button group with string get_value_from_radio_group(radio_group group_to_check);
