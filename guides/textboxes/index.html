Textboxes

Before you begin, make sure your directory has been setup as described in the installation guide.

In this guide we will create two textboxes, one that only allows numbers and one that allows everything to be inputted.
To create a textbox we first need to create a few things. Create the following to get started.

input_area textbox_area = new_input_area(20, 20, 300, 40);   // x, y, width, height
input_area textbox_area_2 = new_input_area(20, 80, 300, 40);
input_font textbox_font = new_input_font("Calibri", 30, COLOR_BLACK);   // font, font size, color
input_border textbox_border = new_input_border(COLOR_BLACK, 5, COLOR_CYAN);   // color, width, selected color
After creating the above artefacts, we need to decide if the textbox on a few properties.
Should the textbox only accept numbers? Do we want the text to be aligned? Do we want a placeholder? How many characters long should the input be?
Once we have the answers to these questions we can create the textbox_text for the textbox and then the textbox itself.
// Number Textbox Example
// Max Length = 2, Placeholder = "Enter a number", Numbers Only = true, Text Alignment = CENTER_ALIGNED
textbox_text textbox_attributes = new_textbox_text(2, "Enter a number", textbox_font, true, CENTER_ALIGNED);
textbox number_textbox = new_textbox("number_textbox", textbox_area, textbox_attributes, textbox_border);

// Normal Textbox Example
// Placeholder = "Enter first name"
textbox_text textbox_attributes_2 = new_textbox_text("Enter first name", textbox_font);
textbox normal_textbox = new_textbox("normal_textbox", textbox_area_2, textbox_attributes_2, textbox_border);
If you are a bit confused about the above functions look at the documentation on the many variations of the same function.

Now that we have our textboxes created we have to draw and update them as follows.
while (!quit_requested())
{
  process_events();
  clear_screen(COLOR_WHITE);
  draw_textbox(number_textbox);
  draw_textbox(normal_textbox);
  update_textbox(number_textbox);
  update_textbox(normal_textbox);
  refresh_screen();
}
You now have a normal textbox and a number textbox fully functional (as long as you followed the guide correctly).
You can access the value of a textbox with textbox_variable.text.value. Returns the value as a string.
