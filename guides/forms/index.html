Forms

Before you begin, make sure your directory has been setup as described in the installation guide.

In this guide we will create a form. A form is essentially a container of buttons, checkboxes, radio button groups and textboxes.
This guide is based off the Guess That Number example.
To get started let's create the textbox and button.

input_area guess_area = new_input_area(100, 70, 400, 40);
input_font guess_style = new_input_font("Alice", 25);
textbox_text guess_text = new_textbox_text(2, "Enter a guess", guess_style, true, CENTER_ALIGNED);
textbox guess_textbox = new_textbox("guess_textbox", guess_area, guess_text);

input_area button_area = new_input_area(200, 140, 200, 40);
input_border button_border = new_input_border(2);
button btn = new_button("submit_button", "Guess", button_area, guess_style, button_border, SUBMIT);
After creating the above artefacts, we can create the form and add the textbox and button to it.
form frm = new_form();
frm.textboxes.push_back(guess_textbox);
frm.buttons.push_back(btn);
Now that we have our form created we have to draw and update it as follows.
The nice thing about forms is we don't have to draw and update everything in it, just the form itself.
while (!quit_requested())
{
  process_events();
  clear_screen(COLOR_WHITE);
  draw_text("Guess The Number!", COLOR_BLACK, "Calibri", 45, 150, 20);
  draw_form(frm);
  update_form(frm);
  refresh_screen();
}
You now know how to create, draw and update forms.
To learn how to do more with forms take a look at the Guess That Number source code.
