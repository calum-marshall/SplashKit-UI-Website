Buttons

Before you begin, make sure your directory has been setup as described in the installation guide.

In this guide we will create a form with a button and a textbox.
Buttons allow users to preform actions within forms (submit and clear).
This guide is based off the Guess That Number example.
To get started let's create the button, form and textbox.

form frm = new_form();

input_area guess_area = new_input_area(100, 70, 400, 40);
input_font guess_style = new_input_font("Calibri", 25);
textbox_text guess_text = new_textbox_text(2, "Enter a guess", guess_style, true, CENTER_ALIGNED);
textbox guess_textbox = new_textbox("guess_textbox", guess_area, guess_text);

input_area button_area = new_input_area(200, 140, 200, 40);
input_border button_border = new_input_border(2);
button btn = new_button("submit_button", "Guess", button_area, guess_style, button_border, SUBMIT);

frm.textboxes.push_back(guess_textbox);
frm.buttons.push_back(btn);
Now that we have our button, form and textbox created we have to draw and update it as follows.
while (!quit_requested())
{
  process_events();
  clear_screen(COLOR_WHITE);
  draw_text("Guess The Number!", COLOR_BLACK, "Calibri", 45, 150, 20);
  draw_form(frm);
  update_form(frm);
  refresh_screen();
}
To check if the form has been submitted (the guess button was clicked) we need to write the following code.
if ((frm.is_submitted || key_typed(RETURN_KEY)) && is_integer(guess))
{
  if (guess_number < the_number)
  {
    message = "The number is higher than " + guess;
    text_color = COLOR_RED;
    clear_form(frm);
  }
  else if (guess_number > the_number)
  {
    message = "The number is lower than " + guess;
    text_color = COLOR_RED;
    clear_form(frm);
  }
  else
  {
    message = "Correct!";
    text_color = COLOR_GREEN;
  }
  frm.is_submitted = false;
}
You now know how to use buttons to manipulate forms.
